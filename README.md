#  Real-Time High-Performance Data Streaming Dashboard

A real-time data visualization dashboard built with **React + Vite + TypeScript + Canvas rendering**, capable of streaming and rendering **10,000+ data points at 60 FPS** with zero-lag interactions.

Designed for live telemetry use-cases such as:

- System & DevOps monitoring
- IoT sensor streams
- Financial tick data dashboards
- Network / cyber-security telemetry
- Real-time industrial/robotics metrics

---

##  Features

   **Real-time data streaming engine**  
   **Canvas-based rendering** for ultra-high FPS  
   Handles **10,000+ points @ 60 FPS**  
   Multiple live charts  
   - Line
   - Bar
   - Scatter Plot
   - Heatmap  
   Virtualized data table for massive datasets  
   FPS + Memory performance monitor  
   Responsive UI layout  
   Optimized with `React.memo`, `useCallback`, `useMemo`

---

##  Tech Stack

| Layer | Technology |
|------|-----------|
Frontend | React, TypeScript, Vite
Charts | Custom Canvas rendering
State | React hooks & performance patterns
Optimization | requestAnimationFrame, memoization
Styling | Tailwind (if applies) / CSS
Data | Fake streaming data generator (simulating live signals)
